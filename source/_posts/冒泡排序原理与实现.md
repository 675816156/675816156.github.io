# 冒泡排序算法描述
![冒泡排序](/images/冒泡排序.png)
# 代码示例
```c
#include <stdio.h>

#define ARRAY_SIZE(a) (sizeof(a)/sizeof(a[0]))

void show_data(char *p, int len)
{
	static int cnt = 0;
	int i = 0;
	printf("%d: ", cnt);
	for (i = 0; i < len; ++i)
		printf("%d ", p[i]);
	printf("\b\n");
	cnt++;
}

void main(void)
{
	int i, j;
	char temp;
	char data[] = {12, 35, 99, 18, 76};

	for (i = 0; i < ARRAY_SIZE(data) - 1; ++i) { // 进行n-1轮比较
		for (j = 0; j < ARRAY_SIZE(data) - i - 1;  ++j) { // 每轮比较前面n-i-1个，因为最后i个已经排序好了
			show_data(data, ARRAY_SIZE(data));
			if (data[j] < data[j + 1]) {
				temp = data[j];
				data[j] = data[j + 1];
				data[j + 1] = temp;
			}
		}
	}
	show_data(data, ARRAY_SIZE(data));
}
```

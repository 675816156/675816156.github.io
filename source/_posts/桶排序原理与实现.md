# 桶排序算法描述
![桶排序](/images/桶排序.png)
# 示例代码
```c
#include <stdio.h>

#define ARRAY_SIZE(a) (sizeof(a)/sizeof(a[0]))

void main(void)
{
	int i, j;
	char array[] = {5, 3, 8, 2, 5};
	char bucket[9] = {0};

	for (i = 0; i < ARRAY_SIZE(array); i++) {
		bucket[array[i]]++;
	}

	printf("从小到大: ");
	for (i = 0; i < ARRAY_SIZE(bucket); i++) {
		if (bucket[i] != 0) {
			for (j = 0; j < bucket[i]; j++)
				printf("%d ", i);
		}
	}
	printf("\b\n");

	printf("从大到小: ");
	for (i = ARRAY_SIZE(bucket); i >= 0; i--) {
		if (bucket[i] != 0) {
			for (j = 0; j < bucket[i]; j++)
				printf("%d ", i);
		}
	}
	printf("\b\n");
}
```
